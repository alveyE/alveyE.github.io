Flipper

End User Documentation

Flipper has a very simplistic interface, which makes it very easy for new users to become familiar with. Whenever a user opens up our application, they are greeted with a very simple home page that prompts the user to type in a product name or scan for a barcode.

When a user types a product into the search bar, Flipper searches Amazon and Ebay to find the price of the product on each of these platforms. It then informs the user of these prices, making this application very effective for users who are trying to ensure they get a good deal. When a user clicks the camera button, Flipper allows the user to scan a barcode and does this same process.

While Flipper searches Amazon and Ebay, it shows the user an activity indicator in order to let them know the search is ongoing. While Flipper is searching, a user can return to the home screen of the application, but can't perform any other actions.

When a user taps the search bar, Flipper displays a list of recent searches. If a recent search is clicked, the application performs the search again and displays up-to-date information to the user.
